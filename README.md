FitNesse fixture to run GUI tests.
Uses https://github.com/RaiMan/SikuliX-API (not google sikuli-api).
For more information about sikuli and what it capable of see sikuli.org

Used in following environment:

- Windows. 
- Java fitlibrary as a test runner

When test is launched for the first time sikuli will add itself to the local path.
Logout and logback in into your machine - test execution should be fine from now on.
